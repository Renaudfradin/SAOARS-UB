# Front SAOARS/UB

## Project Setup

```sh
npm install
```

### Compile for Development

```sh
npm run dev
```

### Environnement Variable.

| Nom de variable            | Valeur                            | Explication                |
| -------------------------- | --------------------------------- | -------------------------- |
| PORT                       | 3000                              | Port du front              |
| VITE_BACKEND_BASEURL       | https://api-saoars.vercel.app     | Port API                   |
